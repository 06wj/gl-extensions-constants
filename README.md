# gl-extensions-constants [![npm][npm-image]][npm-url]
WebGL 1.0 extensions constants.

### Usage

```
var constants = require('gl-extensions-constants');

console.log(constants.COMPRESSED_RGB_S3TC_DXT1_EXT);
```
